# 邵翔 & 周钰瑶 · 婚礼请柬

React + Vite 制作的婚礼电子请柬，包含移动端适配、背景音乐、倒计时、婚礼信息、地图入口、日历入口和 RSVP 表单。

线上地址：

https://shaoxiang880-jpg.github.io/shaoxiang-zhouyuyao-wedding/

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

GitHub Pages 会通过 `.github/workflows/deploy-pages.yml` 自动构建并发布。
